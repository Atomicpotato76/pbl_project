# PBL 모듈 1 로드맵 — 30분 발표용 수정본

문서 상태: `ACTIVE`
수정 기준: 네 개 anchor 보고서 우선, fatal blocker 외 세부 검증은 발표 리스크로 이동
기준 시점: `2026-04-22`

참고: 2차 사이클 이후 전체 공통 로드맵은 `00_공통/00_PBL_메인로드맵_4블록_30분발표.md`를 우선한다. 이 문서는 1차 사이클 결과의 공식 기준으로 유지하되, 30분 발표는 블록 4까지 완료한 뒤 통합한다.

## 1. 프로젝트 목적

N사 수요 증가 작물의 `reference genome(기준 유전체)` 연구 프로젝트를 5주 안에 기획하고, 30분 학부 발표로 설명한다. 이 프로젝트는 실제 wet lab 수행이 아니라 `planning proposal(기획안)`이다. 따라서 완전한 계약 구조, material transfer(시료 이전), ABS benefit-sharing(접근·이익공유) 세부 절차는 발표 본문이 아니라 리스크 관리 항목으로 둔다.

## 2. 현재 진행 상태

| 항목 | 상태 | 설명 |
|---|---|---|
| 시나리오 탐색 | 완료 | 네 개 anchor 보고서로 main/reserve 구조 확정 |
| 작물 후보 | working selection 완료 | Kenya=`Chloris gayana`, India=`Vigna mungo`, Brazil=`Vigna unguiculata`, Morocco=`Lens culinaris` |
| 발표 lead | 선택 필요 | 권장 lead는 Kenya / Rhodes grass |
| 유전체 전략 | 작성 중 | lead scenario에 맞춰 reference upgrade 또는 field-specific variation layer 보강으로 정리 |
| 반복 검증 | 중단 | 치명 오류가 아닌 세부 미확정은 risk/assumption으로 처리 |

## 3. 발표용 전체 흐름

```text
문제상황
  ↓
선정 기준 3축: 수요 경로 / 유전체 명분 / 발표 가능성
  ↓
4개 시나리오 비교: main 2 + reserve 2
  ↓
lead scenario 1개 deep dive
  ↓
reference genome 해독·고도화 전략
  ↓
육종·종자·공공/시장 활용
  ↓
리스크와 결론
```

## 4. 학습목표와 발표 블록 매핑

| PBL 학습목표 | 발표에서 처리하는 방식 |
|---|---|
| 식물종 유전체 구조 | lead crop의 genome size, ploidy(배수성), reference status(기준 유전체 상태)로 설명 |
| 염기서열분석 전략 | long-read(긴 읽기 시퀀싱), Hi-C(염색체 접촉지도), RNA-seq(전사체 시퀀싱) 조합으로 설명 |
| 최신 시퀀싱 방법 | PacBio HiFi / ONT / Hi-C를 “왜 필요한가” 중심으로 설명 |
| 유전체 조립·유전자 예측 | assembly → QC → annotation 흐름으로 설명 |
| 표준유전체 해독과정 | MC2 전체를 3장: 설계, 조립, 활용으로 압축 |
| 팀워크·의사소통 | 역할분담표와 발표 슬라이드 책임으로 처리 |

## 5. Cycle 1 — 작물·시나리오 선정

Cycle 1은 더 이상 open-ended screening(무한 후보 탐색)이 아니다. 네 개 anchor 보고서를 기준으로 다음 상태로 잠근다.

| Scenario | Crop | 발표 상태 | 근거 |
|---|---|---|---|
| Kenya dairy forage resilience | `Chloris gayana` | MAIN / lead 권장 | 낙농 규모, 사료·목초 병목, draft genome 고도화 여지 [A-KENYA][W-KENYA] |
| India protein-security food-legume public route | `Vigna mungo` | MAIN / second | pulses 정책경로, Urad-specific procurement signal [A-INDIA][W-INDIA] |
| Brazil PNAE/PAA family-farming public procurement | `Vigna unguiculata` | RESERVE | 가족농 공공조달 경로가 분명함 [A-BRAZIL][W-BRAZIL] |
| Morocco structural water-stress adaptation | `Lens culinaris` | RESERVE | 구조적 물 부족과 rainfed food legume 논리 [A-MOROCCO][W-MOROCCO] |

## 6. Cycle 2 — 유전체 해독 설계

발표에서는 lead scenario 1개를 기준으로 Cycle 2를 깊게 전개한다. 권장 lead가 Kenya / Rhodes grass일 때 흐름은 다음과 같다.

1. `Genome status audit(유전체 상태 점검)`: 기존 draft genome의 assembly size, N50, BUSCO, chromosome anchoring 여부를 정리한다.
2. `Reference build strategy(기준 유전체 구축 전략)`: long-read + Hi-C로 chromosome-scale assembly를 목표로 한다.
3. `QC and annotation(품질평가와 유전자 주석)`: BUSCO, LAI, read mapping, RNA-seq evidence를 사용한다.
4. `Variation layer(변이 층)`: cultivar/accession별 resequencing으로 drought/salt/temperature resilience 관련 변이를 해석한다.
5. `Breeding bridge(육종 연결)`: forage yield, persistence(지속성), stress tolerance를 seed pathway와 연결한다.

## 7. 30분 시간 배분

| 블록 | 시간 | 핵심 메시지 |
|---|---:|---|
| 문제상황 | 3분 | 왜 “수요 증가 작물 + 기준 유전체”인가 |
| 선정기준 | 4분 | demand route, genome rationale, feasibility 세 축 |
| 시나리오 비교 | 4분 | main/reserve 구조를 한 장으로 표시 |
| lead scenario | 9분 | 시장, 작물, 유전체 공백, 왜 지금 필요한지 |
| 해독 전략 | 6분 | sequencing → assembly → QC → annotation |
| 활용·리스크 | 4분 | breeding, seed pathway, adoption, regulation |

## 8. Decision Gate 수정

기존 문서의 gate는 너무 엄격했다. 이제 gate는 세 단계만 둔다.

| Gate | 판정 기준 | 발표에서의 처리 |
|---|---|---|
| Fatal blocker | 공식 정책·법·유전체 사실이 anchor와 정면충돌해 발표 논리가 붕괴 | 즉시 수정 |
| Material risk | buyer, legal route, seed pathway가 미완성이나 발표 가능 | 리스크 슬라이드로 이동 |
| Detail debt | N50 세부값, accession 세부정보, 계약 구조 등 | 백업 Q&A로 이동 |

## 9. 발표 lead 권장

현재 30분 발표의 lead는 `Kenya / Chloris gayana`가 가장 깔끔하다. 이유는 세 가지다.

1. downstream market(하위 시장)이 낙농으로 명확하다.
2. forage bottleneck(사료·목초 병목)이 문제상황과 직접 연결된다.
3. reference genome이 완전히 포화된 작물이 아니므로 표준유전체 고도화 명분이 강하다.

India / Vigna mungo는 정책·시장 서사가 강하지만 공개 reference가 빠르게 고도화되어 “왜 추가 genome work가 필요한가”를 조금 더 정교하게 말해야 한다. Brazil과 Morocco는 reserve slide로 충분하다.

## 고난도 실무 용어만 설명

- `chromosome-scale assembly`: 조립 결과가 염색체 단위에 가깝게 배열된 상태. 발표에서는 표준유전체 설립의 현실적 목표로 사용한다.
- `haplotype`: 함께 유전되는 변이 묶음. 병저항성이나 stress-response 형질을 설명할 때 중요하다.
- `structural variation, SV`: 큰 DNA 구간의 삽입·결실·역위·중복. 단순 SNP보다 형질 영향이 클 수 있다.
- `pangenome`: 여러 계통의 공통·가변 유전체 정보를 함께 담은 참조 체계. 본문 핵심이 아니라 확장 카드로만 사용한다.
