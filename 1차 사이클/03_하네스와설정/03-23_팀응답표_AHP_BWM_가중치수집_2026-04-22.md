# 2026-04-22 팀 응답표 - AHP/BWM 가중치 수집용

## 사용 목적

- 이 문서는 `축 가중치`만 수집한다.
- 이번 단계에서는 `작물별 세부 점수`를 넣지 않는다.
- 지금 회수하는 값은 이후 `TOPSIS` 적용 시 `criterion weight` 입력값으로 사용한다.
- 작물별 성능 점수 행렬은 별도 세션에서 확정한 뒤, 그다음 TOPSIS에 결합한다.

## 공통 점수축

| 점수 | 의미 |
|---|---|
| `1` | 두 축이 동등하게 중요 |
| `2` | `1`과 `3`의 중간 |
| `3` | 한 축이 다른 축보다 약간 더 중요 |
| `4` | `3`과 `5`의 중간 |
| `5` | 한 축이 다른 축보다 명확히 더 중요 |
| `6` | `5`와 `7`의 중간 |
| `7` | 한 축이 다른 축보다 매우 더 중요 |
| `8` | `7`과 `9`의 중간 |
| `9` | 한 축이 다른 축보다 극도로 더 중요 |

메모:

- `AHP`에서는 `더 중요한 축`과 `강도(1~9)`를 같이 적는다.
- `BWM`에서는 `Best -> Others`, `Others -> Worst` 비교에 같은 `1~9` 강도를 쓴다.
- 배경 설명은 구두로 하고, 이 문서에는 `선택값`만 남기는 용도로 쓴다.

## 적용 순서 메모

1. 팀원별로 `AHP` 또는 `BWM` 응답표를 회수한다.
2. 회수한 값을 바탕으로 축 가중치를 계산한다.
3. `AHP 결과`와 `BWM 결과`를 비교한 뒤 최종 축 가중치를 잠근다.
4. 별도 세션에서 작물별 성능 점수 행렬을 확정한다.
5. 마지막에 `최종 가중치 + 작물별 성능 점수`를 결합해서 `TOPSIS`를 적용한다.
