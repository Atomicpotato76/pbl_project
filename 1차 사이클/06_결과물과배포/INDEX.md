# 배포세트 인덱스

이 폴더는 작업용 원문이 아니라, 외부 전달이나 검증용으로 복사해 둔 첨부 세트 모음이다.
원문 기준은 `최신 상황` 또는 `시나리오 검증용`에서 확인한다.
중복 문서의 기준본 위치는 `운영지원_통합/01_문서안내허브/중복문서_기준본_매핑.md`를 우선 본다.
이 폴더 안 문서의 `구 파일 / 신 파일 / 공통 입력 / 참고보관` 분류와 번호 체계는 `00_배포문서_분류및권장명.md`를 먼저 본다.

## 먼저 보기

- `00_배포문서_분류및권장명.md`: 진행 순서, 분류, 한글 통일명을 한 번에 정리한 지도

## 패키지별 용도

### `10_구기준_기본입력세트`

- 구기준 기본 입력 세트
- 포함 파일:
  - `00-01_문제정의_PBL원문.md`
  - `10-01_구기준_작물선정통합체크리스트.md`
  - `10-02_구기준_시나리오선정기준_초안.md`
  - `10-03_구기준_시나리오선정기준_수정본.md`
  - `12-01_전환자료_시나리오실패사례.md`

### `11_구결과_스크리닝세트`

- 구기준 결과를 보강할 때 붙이는 세트
- 포함 파일:
  - `11-01_구결과_작물스크리닝결과.md`

### `12_구기준_확장검증세트`

- 구기준 스트레스 테스트용 확장 세트
- 포함 파일:
  - `00-01_문제정의_PBL원문.md`
  - `10-01_구기준_작물선정통합체크리스트.md`
  - `10-02_구기준_시나리오선정기준_초안.md`
  - `11-01_구결과_작물스크리닝결과.md`
  - `12-01_전환자료_시나리오실패사례.md`

### `20_신기준_시나리오검증세트`

- 신기준 시나리오 검증용 핵심 세트
- 포함 파일:
  - `00-01_문제정의_PBL원문.md`
  - `20-01_신원칙_PBL우선원칙과추가시나리오.md`

  - `50-01_조사결과_시나리오병렬조사.md`
  - `60-01_판정결과_시나리오축소검증.md`

### `90_참고보관_로드맵세트`

- 직접 판단보다 역사 복기용 보관 세트
- 포함 파일:
  - `90-01_참고보관_프로젝트전체로드맵.md`

## 사용 원칙

1. 새 첨부 세트는 이 폴더 아래에만 만든다.
2. 파일을 수정해야 하면 먼저 원문 위치를 찾아서 수정하고, 필요할 때만 여기 복사본을 갱신한다.
3. `21-01_신기준_시나리오선정기준_강화본.md`의 활성 기준본은 `최신 상황/01_프로젝트기반과활성기준/01-04_활성기준_시나리오선정기준_v1.2.md`다.
4. `03_배포세트` 안에는 하네스 원문과 피드백 로그가 없으므로, 그 문서를 찾을 때는 `최신 상황` 또는 `운영지원_통합/02_작업기록`으로 이동한다.
