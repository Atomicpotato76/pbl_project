## 한 줄 판정

현재 `N1 x Chloris gayana`, `N2 x Lens culinaris`, `N2 x Vigna mungo` 세 lane(레인, 후보 경로) 모두 `carry-forward 가능한 provisional lane(계속 들고 갈 수 있는 임시 후보 경로)`이지만, **세 lane 모두 아직 `score-ready(점수화 가능 상태)`는 아니다.**

즉 지금 단계의 정확한 판정은:

- `live scope(현재 실제 범위)`는 유지
- `lane survival(레인 생존)`은 유지
- `scoring eligibility(점수화 자격)`는 전부 미충족
- `scenario close(시나리오 잠금)`도 아직 미완료
- 따라서 `crop selection(최종 작물 선정)` 단계로는 아직 못 넘어감

---

## 현재 닫힌 것

### 1. live scope는 더 이상 넓히지 않는다

- 현재 live scope는 `N1 x Chloris gayana`, `N2 x Lens culinaris`, `N2 x Vigna mungo`만 유지한다.
- 이전 `legacy ranking(과거 순위 메모)`이나 reserve lane(예비 레인)은 이번 live run(현재 실제 실행) 판단에 carry하지 않는다.

### 2. 세 lane 모두 outright fail(즉시 탈락) 상태는 아니다

- `N1`은 Kenya 쪽 `legal route skeleton(법 경로의 기본 골격)`과 `formal seed route(공식 종자 경로)`가 둘 다 보인다.
- `N2-Lens`는 lentil-specific named line(렌틸 이름이 직접 보이는 조달/판매 라인)이 아예 없는 것은 아니다.
- `N2-Urd`는 buyer-side signal(구매자 쪽 신호)만 놓고 보면 `N2-Lens`보다 조금 더 직접적이다.

즉 셋 다 지금은 `fail(탈락)`이 아니라 `pending-gate(게이트 미종결)`이다.

### 3. 방법론(methodology, 판정 방식) 정렬은 좋아졌다

- `HF-0 / HF-3 / HF-4 / HF-7`를 `pre-BWM hard gate(점수화 이전의 강한 통과 조건)`로 유지했다.
- `G2`와 `G8`을 섞지 않고 따로 보려는 방향이 유지됐다.
- `existing public resource(기존 공개 자원)` 존재 여부와 `genome necessity(새 유전체 필요성)`를 자동 연결하지 않았다.

이건 꽤 중요하다. 지금은 적어도 “route가 좋아 보이니까 그냥 점수 매기자” 식으로 흐르지는 않고 있다.

## 아직 안 닫힌 것

### 1. 세 lane 모두 score-ready가 아니다

`I3`와 `R1`이 같이 가리키는 결론은 같다.

| Lane | 현재 상태 | score-ready 여부 | 핵심 이유 |
|---|---|---|---|
| `N1 x Chloris gayana` | `pending-gate` | `no` | public genome insufficiency 미증명, G6 conflict 미종결, G2/G8 미분리 |
| `N2 x Lens culinaris` | `pending-gate` | `no` | buyer lock 미종결, public Lens resource 이후 insufficiency 미증명, legal/value-capture 미종결 |
| `N2 x Vigna mungo` | `pending-gate` | `no` | buyer/program은 더 직선적일 수 있으나 genome necessity와 legal/value-capture 미종결 |

### 2. hard gate가 전부 열려 있다

세 lane 공통으로 아직 안 닫힌 축은 거의 같다.

- `HF-0`: 왜 지금 새 reference genome(기준 유전체)이 필요한가
- `HF-3`: 기존 공개 자원 위에서 novelty(신규성)를 어떻게 방어할 것인가
- `HF-4`: genome-to-breeding bridge(유전체에서 육종/활용으로 이어지는 다리)가 실제로 있는가
- `G2`: payer / budget holder / procurement buyer가 분리되어 있는가
- `G8`: technical user / delivery controller / transaction object / acceptance criterion이 분리되어 있는가

즉, 지금 막히는 이유는 “아예 길이 없어서”가 아니라, **길은 있는데 그 길이 실제로 왜 필요한지와 누가 실제로 받아 쓰는지가 안 잠겨서**다.

## Lane별 통합 판정

## N1 x Chloris gayana

### 현재 읽기

- 살아 있는 lane이다.
- Kenya 쪽 `official route components(공식 경로 구성요소)`와 `seed route(종자 경로)`는 보인다.
- 하지만 `existing public genome exists(기존 공개 genome이 이미 존재함)` 이후에 왜 새 genome이 필요한지가 아직 안 닫힌다.

### 핵심 blocker

- `G6`: `Kenya ABS 2025`와 `NEMA` 포털/가이드 문구 사이 `official-source conflict(공식 출처 충돌)`
- `HF-0 / HF-3 / HF-4`: benchmark use-case(판정 기준이 되는 실제 쓰임새) 기준 insufficiency memo(불충분성 메모) 부재
- `G5 / G8`: named multiplication/distribution/adoption owner와 technical user 분리 미완료
- `G2`: payer / budget / procurement 구조 미분리

### 현재 verdict(판정)

`N1`은 `drop(탈락)`이 아니라 `reopen-required carry lane(다시 열어야 하지만 계속 들고 가는 레인)`이다.

## N2 x Lens culinaris

### 현재 읽기

- lentil-specific route는 있다.
- 하지만 `national lentil buyer lock(전국 단위 렌틸 구매자 고정)`은 아직 아니다.
- 공개 Lens resource가 이미 두꺼워서, genome-core burden(유전체 필요성 부담)은 가볍지 않다.

### 핵심 blocker

- `G2 / G8`: crop-specific buyer/payer/technical-user 분리 미완료
- `HF-0 / HF-3 / HF-4`: existing Lens stack(기존 렌틸 공개 자원 묶음) 이후 insufficiency 미증명
- `G4`: import dependence / intervention sensitivity(수입 의존 / 개입 정책 민감도) 정량화 부족
- `G5 / G6`: value-capture와 legal route의 material-level closure(개별 자원 수준 잠금) 미완료

### 현재 verdict

`N2-Lens`도 살아 있지만, 지금 당장 점수화할 수 있는 lane은 아니다.

## N2 x Vigna mungo

### 현재 읽기

- buyer-side signal만 보면 `N2-Lens`보다 조금 더 직접적이다.
- 하지만 그 우위는 buyer-side에 한정된다.
- genome-core burden과 legal/value-capture burden은 여전히 무겁다.

### 핵심 blocker

- `HF-0 / HF-3 / HF-4`: public Vigna mungo genome 이후 necessity/novelty/bridge 미종결
- `G2 / G8`: buyer/program은 보이지만 exact field split은 아직 미완료
- `G6`: accession-level provenance(개별 자원 출처 계보), notified status, MLS practicality(실제 MLS 활용 가능성) 미종결
- `G5`: value-capture path가 아직 강하게 잠기지 않음

### 현재 verdict

`N2-Urd`는 `provisional buyer-side carry signal(구매자 쪽에서만 더 나은 임시 신호)`은 있지만, overall lane close(전체 레인 잠금) 수준은 아니다.

## 지금 절대로 하면 안 되는 것

아래 4개는 아직 금지 상태로 읽는 게 맞다.

1. `BWM scoring(점수화)` 진입
2. `N2-Lens vs N2-Urd` 최종 winner 선언
3. `N1 vs N2` 최종 우열 선언
4. `final crop recommendation(최종 작물 추천)` 선언

이 네 가지를 지금 해버리면, 현재 남아 있는 `pending-gate`를 soft score(가벼운 점수 항목)처럼 다루게 된다.
