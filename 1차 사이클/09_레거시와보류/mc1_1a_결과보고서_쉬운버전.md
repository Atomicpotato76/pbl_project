# MC1-1a 결과 보고서 — 형주가 읽는 버전

> 작성: 2026-04-20 | 단계: MC1-1a (SURVEY) 완료 → Gate PASS
> 이 문서는 "지금까지 뭘 했고, 뭘 기준으로 판단했고, 뭐가 나왔는지"를 정리한 중간 보고서야.
> 최종 전문 보고서는 MC1-1c 끝나고 한번에 뽑을 예정.

---

## 1. 우리가 지금 하고 있는 일

N사(종자회사)가 표준유전체 해독 프로젝트를 하려고 해. **"어떤 작물을 할까?"**를 정하는 게 MC1-1의 목표야.

후보가 19개에서 9개로 줄었고, 이 9개를 실제 데이터로 검증하는 중이야. MC1-1은 3단계로 나뉘어:

| 단계 | 하는 일 | 상태 |
|---|---|---|
| **MC1-1a** (SURVEY) | 9개 작물의 시장·지정학·소비 등 실제 데이터 조사 | ✅ 완료 |
| **MC1-1b** (EVALUATE) | 데이터 기반으로 점수 다시 매기기 + 흔들기 테스트 | ⬜ 다음 |
| **MC1-1c** (SYNTHESIS) | Top 3 중 최종 1개 확정 + 발표 자료 작성 | ⬜ 대기 |

## 2. 점수 체계 — 이걸로 작물을 비교해

### D점수 5개 (각 작물마다 0/1/2점)

| D항목 | 쉽게 말하면 | 예시 |
|---|---|---|
| **D1** 시장 매력도 | "이 작물 종자 팔면 돈 되나?" | 시장 크기, 성장률, hybrid 비중 |
| **D2** 유전체 신규성 | "표준유전체 해독하면 새로운 가치가 있나?" | 이미 잘 해독된 작물이면 가치↓ |
| **D3** 사업 실행 가능성 | "N사가 실제로 할 수 있나?" | IP 보호, 현지 파트너, 규제 환경 |
| **D4** 기술 장벽 | "유전체가 너무 복잡하지 않나?" | 게놈 크기, 배수체 여부 |
| **D5** 전략적 타이밍 | "지금이 적기인가?" | 정책 지원, 시장 진입 window |

각 D항목의 0/1/2는 정해진 기준표(루브릭)가 있어. 예를 들어 D1=2를 받으려면 "글로벌 종자시장 ≥$500M 또는 CAGR ≥3%, hybrid 비중 ≥30%"를 만족해야 해.

### 가중치 — "D끼리 중요도가 다르다"

모든 D가 동등하진 않아. AHP라는 방법으로 중요도를 정했는데, **결과만 기억하면 돼**:

| D항목 | 가중치 | 해석 |
|---|---|---|
| D1 (시장) | **0.363** (36.3%) | 가장 중요 — 돈이 안 되면 사업 의미 없으니까 |
| D2 (신규성) | 0.219 (21.9%) | 둘 다 동일 — 연구 가치 + 실행력 |
| D3 (실행) | 0.219 (21.9%) | ↑ 위와 동일 |
| D5 (타이밍) | 0.107 (10.7%) | 타이밍은 중요하지만 시장·실행보다는 덜 |
| D4 (기술) | 0.092 (9.2%) | 가장 낮음 — 기술은 돈 쓰면 해결 가능성 높아서 |

### 최종 점수 산식

```
Score = (D1×0.363 + D2×0.219 + D3×0.219 + D4×0.092 + D5×0.107) ÷ 2 × 100
```

이게 뭘 하는 거냐면: 각 D점수에 중요도를 곱해서 합산 → 만점(2.0) 기준으로 100점 만점 환산. **숫자가 높을수록 좋은 후보**.

## 3. MC1-1a에서 뭘 조사했나

GPT Pro에게 9개 작물의 **5개 축** 데이터를 찾아오라고 시켰어:

| 축 | 내용 | 이게 어떤 D에 영향 주나 |
|---|---|---|
| 축1: 시장 데이터 | 생산량, CAGR, 종자시장 규모 | → D1 (시장), D5 (트렌드) |
| 축2: 소비구조 | 어디서 누가 먹나, 수요 패턴 | → D1 (보조 근거) |
| 축3: 지정학 리스크 | 규제, ABS(나고야의정서), IP 보호 | → D3 (사업 실행) |
| 축4: 사회문화 요인 | 농가 채택 가능성, 문화적 선호 | → D3 (보조 근거) |
| 축5: 정책 동향 | VACS, CGIAR 2025, 국가 종자정책 | → D5 (타이밍) |

**조사 깊이는 두 단계**로 나눴어:

- **Tier A (심화)**: 비둘기콩, 고추, 배추 — 현재 Top 3이니까 꼼꼼히
- **Tier B (스캔)**: 시금치, 들깨, 참깨, 녹두, 알팔파, 라이그라스 — 핵심 지표만 빠르게, 역전 가능성 보이면 심화

그리고 D2(유전체 신규성)와 D4(기술 장벽)는 시장 조사로 알 수 없는 영역이라 MC1-1a 범위 밖으로 뺐어. 이건 기존 점수를 유지하고, 나중에 MC2-1에서 기술 검증할 때 다시 봐.

## 4. 데이터에 붙인 "신뢰도 라벨"

GPT가 찾아온 데이터가 다 같은 수준이 아니거든. 그래서 각 셀마다 이런 라벨을 붙였어:

| 라벨 | 뜻 | 신뢰도 |
|---|---|---|
| [실측] | FAOSTAT, UN Comtrade 같은 공식 통계 | 가장 높음 |
| [시장추정] | 시장 보고서 기반 추정 | 높음 |
| [문헌확인] | 범위 밖 항목을 논문/DB로 확인 | 보통 |
| [전문가추론] | AI가 도메인 지식으로 추론 | 낮음 |
| [미확보] | 데이터 없음 | 위험 |

**Gate 통과 조건에서 중요한 건 [미확보]가 얼마나 적은가야.** [실측]+[시장추정]이 많을수록 믿을 만한 판단이 되는 거지.

## 5. GPT Pro 조사 결과 — 핵심 발견

GPT Pro가 26분 42초 동안 생각하고 내놓은 결과야. 작물별로 정리하면:

### 🌶️ 고추 — "가장 안정적인 후보"

- 글로벌 생산량 38.31 Mt, CAGR +1.45%
- Hybrid 시장이 확고해서 종자회사가 IP 회수하기 좋음
- **D1=2 유지, D3=2 유지, D5=2 유지** — 모든 축에서 실증 데이터가 탄탄
- 한마디로: 뭘 흔들어도 잘 안 바뀜. "안전한 선택"

### 🥬 배추 — "성장률은 낮지만 전략적 가치"

- 전체 시장 성장률 자체는 높지 않아
- 근데 김치 프리미엄 세그먼트 + 아시아 전략 시장이라는 각도로 보면 D1=2 정당화 가능
- **D1=2 유지(논리 변경 필요), D3=2 유지, D5=2 유지**
- 한마디로: "성장률로 밀기보다 전략적 프리미엄으로 밀어야 함"

### 🫘 비둘기콩 — "가장 논쟁적"

- India thesis가 핵심: 인도 시장에서 hybrid F1으로 승부
- **D5를 1→2로 올려야 함**: VACS/CGIAR이 pigeon pea를 6개 시장 세그먼트로 통합 추진 중 → 타이밍 좋음
- **D3를 2→1로 내려야 함**: 인도 PPV&FR법에서 농민 종자 권리가 강해서 F1 hybrid IP 회수가 구조적으로 어려움 + 비공식 종자 유통 시스템이 큰 장벽
- 한마디로: "정책 타이밍은 좋은데, 실제로 사업하기는 생각보다 어려울 수 있다"

### 🥬 시금치 — "가장 강한 역전 후보"

- **CAGR +3.20%로 9작물 중 최고**
- Baby leaf, fresh-cut, 식물공장(CEA), 건강편의식 트렌드가 전부 몰려있음
- D1도 D5도 과소평가됐을 가능성 → **D5=1→2 검토 대상**
- 한마디로: "지금 Top 3 밖인데, 데이터 보니까 Top 3에 들어갈 자격이 충분할 수도"

### 나머지 (참깨, 녹두, 들깨, 알팔파, 라이그라스)

- **참깨**: VACS/CGIAR 정책 타이밍으로 D5 상향 여지 있지만, D1·D3가 약해서 즉시 역전 어려움
- **녹두**: commercial seed penetration이 불명확 → D3 보수화 필요
- **들깨**: 한국 전용이면 D3=2 유지, 글로벌이면 D3=1
- **알팔파·라이그라스**: 시장 규모 자체는 크지만, N사의 F1 hybrid / 표준유전체 사업 모델이랑 안 맞음 → **사실상 탈락**

## 6. Gate Check — "MC1-1b로 넘어가도 되나?"

MC1-1a 결과가 충분한지 3개 조건으로 확인했어:

### 조건 1: 핵심 데이터가 빠진 칸이 3개 이하인가?

검사 범위: Top 3(비둘기콩, 고추, 배추) × MC1-1a 범위 내 D항목(D1, D3, D5) = **9칸**

| | D1 | D3 | D5 |
|---|---|---|---|
| 비둘기콩 | ✅ 실측/시장추정 | ✅ 실측/문헌확인 | ✅ 실측 (VACS) |
| 고추 | ✅ 실측 (FAO) | ✅ 실측/시장추정 | ✅ 실측 |
| 배추 | ✅ 실측 (FAO) | ✅ 실측/시장추정 | ✅ 시장추정 |

[미확보] = **0개** (기준: ≤3개) → ✅ **PASS**

> 참고: D2, D4는 MC1-1a 범위 밖이라 Gate 카운트에서 빠져. 이것들까지 넣으면 "데이터 없어서 구조적으로 통과 불가"가 되거든 — 이 문제를 v2.2에서 잡았어.

### 조건 2: 믿을 만한 데이터가 60% 이상인가?

Top 3의 9칸 중에서 [실측] 또는 [시장추정]인 칸 비율:

- 9칸 중 8~9칸이 [실측] 또는 [시장추정] → 약 **89~100%** (기준: ≥60%)
- 비둘기콩 D3만 proxy 혼합이 좀 껴있지만 [문헌확인] 범위

→ ✅ **PASS**

### 조건 3: "혹시 Top 3에 끼어야 할 작물"을 놓치진 않았나?

GPT가 Tier B 6개 전부 스캔하고 역전 가능성을 판별했어:

- **시금치 = 가장 강한 역전 후보** (CAGR 최고 + 트렌드 복합 + D5 과소평가)
- 참깨 = 정책 타이밍은 좋지만 즉시 역전은 어려움
- 알팔파/라이그라스 = N사 사업모델 fit 낮아 탈락
- 녹두/들깨 = 상업적 종자 시장 불명확

→ ✅ **PASS** (역전 후보 식별 완료)

### Gate 종합: 3/3 **PASS** → MC1-1b 진입 가능

## 7. MC1-1b에서 반영해야 할 것들

Gate는 통과했지만, MC1-1b에서 점수를 다시 매길 때 아래 4가지를 꼭 반영해야 해:

| # | 뭘 바꿔야 하나 | 왜 |
|---|---|---|
| 1 | **비둘기콩 D3: 2→1 하향** | 인도 PPV&FR 농민 종자 권리 + 비공식 종자 유통 → F1 IP 회수 구조적 어려움 |
| 2 | **비둘기콩 D5: 1→2 상향 검토** | VACS/CGIAR pigeon pea 6개 시장 세그먼트 통합 추진 |
| 3 | **시금치 재평가** | CAGR +3.20% 최고, D1·D5 둘 다 과소평가 가능성 → Top 3 진입 가능성 |
| 4 | **배추 D1 논리 변경** | "성장률 높아서"가 아니라 "김치 프리미엄 세그먼트 전략적 가치"로 정당화 |

추가로 참깨 D5=1→2도 검토 대상이야 (VACS/CGIAR sesame 3개 market segment 통합).
