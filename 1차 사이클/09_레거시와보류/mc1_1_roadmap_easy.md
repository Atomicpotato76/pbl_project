# MC1-1 로드맵 — 형주용 요약 (v2.3 기준)

> 이 문서는 `mc1_1_detailed_roadmap.md` v2.3의 **핵심만 추린 요약본**이야.
> 전문 용어는 "이게 뭐고, 어디에 쓰이는지"만 설명했어.

---

## 한줄 요약

**"19개 작물 중에서 N사가 표준유전체 해독할 작물 1개를 고르는 과정"**을 3단계로 쪼갠 계획서야.

---

## 전체 흐름 (3단계)

```
MC1-1a (조사)  →  🚦  →  MC1-1b (채점)  →  🚦  →  MC1-1c (최종 선정)
데이터 모으기        충분한가?    점수 다시 매기기     Top 3 OK?    1개 고르기 + 발표 준비
```

🚦 = **Gate** (관문). 조건 충족해야 다음 단계로 넘어가.

---

## MC1-1a: 데이터 모으기 (SURVEY)

**뭘 하는 단계?** Shortlist 9개 작물의 "진짜 데이터"를 모으는 거야.

기존에 GPT한테 물어서 점수 매긴 게 있는데, 그건 일반 지식이잖아. 이번에는 **실제 통계·논문·정책 자료**로 뒷받침하는 거지.

### 5개 축 (= 조사할 카테고리)

| 축 | 뭘 보는 건지 | 예시 |
|---|---|---|
| 축1: 시장 | 이 작물 종자시장이 얼마나 큰지 | FAOSTAT 재배면적, 종자시장 규모 |
| 축2: 소비 | 누가, 어떻게 소비하는지 | fresh vs 가공, 지역별 수요 차이 |
| 축3: 지정학 | 종자 수출입에 법적·정치적 장벽이 있는지 | India 규제, ABS/나고야 의정서 |
| 축4: 사회문화 | 농가가 실제로 이 종자를 살 건지 | farmer-saved seed 비율, 품종 전환 의향 |
| 축5: 정책 | 정부·국제기구가 밀어주는 작물인지 | VACS 포함 여부, CGIAR 우선순위 |

### 조사 우선순위

- **Tier A** (깊이 조사): 비둘기콩, 고추, 배추 — 현재 Top 3이니까 꼼꼼하게
- **Tier B** (스캔): 나머지 6개 — 핵심 지표만 빠르게. 역전 가능성 보이면 깊이 전환

### 근거 라벨 (데이터 출처 표시)

각 데이터마다 "어디서 왔는지" 태그를 붙여:

| 태그 | 의미 |
|---|---|
| [실측] | FAOSTAT, UN Comtrade 같은 공식 통계 |
| [시장추정] | 시장조사 보고서 기반 추정 |
| [전문가추론] | LLM이 도메인 지식으로 추론한 것 |
| [미확보] | 데이터 못 찾음 |
| [문헌확인] | D2/D4처럼 이 단계 범위 밖인데, 논문으로 확인한 것 |

### Gate: 데이터 충분한가?

MC1-1b로 넘어가려면 3개 조건을 통과해야 해:

1. **[미확보] 셀이 3개 이하** — Top 3 작물 × D1/D3/D5 (총 9칸) 중에서
2. **[실측]+[시장추정]이 60% 이상** — 추론에만 의존하면 안 됨
3. **Tier B 역전 신호 확인 완료** — 혹시 Top 3 뒤집을 작물 없는지 체크했는지

## MC1-1b: 점수 다시 매기기 (EVALUATE)

**뭘 하는 단계?** MC1-1a에서 모은 데이터로 D1~D5 점수를 다시 매기는 거야.

### D1~D5가 뭔데?

작물을 평가하는 5가지 기준이야:

| 기준 | 한줄 설명 | 어디서 쓰여? |
|---|---|---|
| D1: 시장 매력도 | 종자시장이 크고 성장하는가? | 축1, 축2 데이터로 검증 |
| D2: 유전체 신규성 | 새 유전체 해독할 가치가 있는가? | 이미 있는 문헌으로 확인 (이 단계 범위 밖) |
| D3: 사업 실행 가능성 | N사가 실제로 할 수 있는가? | 축3, 축4 데이터로 검증 |
| D4: 기술 장벽 | 시퀀싱이 기술적으로 가능한가? | 이미 있는 문헌으로 확인 (이 단계 범위 밖) |
| D5: 전략적 타이밍 | 지금이 진입 적기인가? | 축5, 축1 트렌드로 검증 |

각 D에 0/1/2점을 줘. 루브릭(채점 기준표)이 전문 로드맵에 있어.

### AHP Weight — 어디에 쓰이나?

> **핵심**: 5개 기준(D1~D5)이 **다 같은 중요도가 아니야**. Weight는 "이 기준이 얼마나 중요한지"를 숫자로 표현한 거야.

| 기준 | Weight | 해석 |
|---|---|---|
| D1 (시장) | 0.363 | **제일 중요** — 종자회사니까 시장이 1순위 |
| D2 (유전체) | 0.219 | 두 번째 |
| D3 (실행가능) | 0.219 | D2와 동급 |
| D5 (타이밍) | 0.107 | 중요하긴 한데 덜 |
| D4 (기술장벽) | 0.092 | 가장 덜 중요 — 기술은 해결 가능하니까 |

**Weight를 어디에 쓰냐?** → 최종 점수를 계산할 때 D점수에 곱해. 시장이 좋은 작물(D1 높음)이 전체 점수에 더 크게 반영되도록.

### Sensitivity Analysis — 어디에 쓰이나?

> **핵심**: "weight를 살짝 바꿔봤을 때 순위가 뒤집히는가?"를 테스트하는 거야.

예를 들어 D1의 weight를 20% 올려봤더니 비둘기콩이 1위에서 3위로 떨어지면? → 비둘기콩 1위는 "D1이 0.363이라는 가정"에 기대고 있다는 뜻이야. 그 가정이 흔들리면 결론도 흔들려.

이걸 6개 시나리오로 돌려:
1. D1 weight +20%
2. D1 weight -20%
3. D2 > D1 역전
4. D3/D4/D5 weight ±20%
5. 미결 셀 비관(0점) 처리
6. 미결 셀 낙관(2점) 처리

### 역전이 발생하면? (v2.3 신규 규칙)

아래 3가지를 체크해:

| 기준 | 뭘 보는 건지 |
|---|---|
| ① 현실적 trigger | N사 전략이 바뀔 이유가 있어서 weight가 실제로 변할 수 있나? |
| ② 범위 내 발생 | ±20% 안에서 역전이 일어나나, 아니면 ±50%까지 가야 일어나나? |
| ③ 역전 마진 | 역전 후 점수 차이가 크나 작나? |

→ 3개 중 2개 이상 "높음"이면 그 후보를 진지하게 재검토. 아니면 기존 순위 유지.

### Gate: Top 3 확정

1. 점수 상위 3개
2. 동점이면 D1 → D3 순으로 우선
3. 미결 셀 있는 작물은 비관 시나리오에서도 Top 3여야 포함
4. Sensitivity 역전은 위의 판정 규칙 적용

## MC1-1c: 최종 1개 고르기 + 발표 준비 (SYNTHESIS)

**뭘 하는 단계?** Top 3를 깊이 비교해서 1개를 최종 선정하고, 발표 블록 1 brief를 쓰는 거야.

### 비교하는 4가지

1. **N사 사업 적합도** — hybrid F1 수익 구조, seed system
2. **유전체 프로젝트 적합도** — 학부 3학년이 5주 기획으로 보여줄 수 있나?
3. **리스크** — 지정학/기술/시장 리스크 각각 순위
4. **발표 매력도** — 5분 안에 "왜 이 작물?"을 설득력 있게 말할 수 있나?

### 발표 블록 1 구조 (5분)

1. 도입: 왜 N사가 이 프로젝트를 하는가 (30초)
2. 선정 과정: 19개 → Top 3 → 1개 (1분)
3. 최종 작물 소개 (2분)
4. 핵심 수치 3개 (1분)
5. 다음 단계 예고 (30초)
