# N1 Crop-Group Long List Draft v1

## 상태

이 문서는 `formal activation 전 draft`다.  
`N1`의 `SG-6`, `SG-8`이 아직 완전히 잠기지 않았으므로, 이 long list는 **준비용 draft**로만 쓴다.

## 전제 시나리오

`N1 narrowed expression`

`formal seed-propagated Chloris pasture grasses for Kenya mixed-ruminant/dairy-adjacent systems`

## carry 원칙

이 draft에는 아래 조건을 동시에 만족하는 후보만 올린다.

- Kenya official variety/source 문서가 있음
- formal seed route를 설명할 수 있음
- public genome saturation이 Napier/Urochloa보다 낮음
- mixed-ruminant / dairy-adjacent forage use-case를 적을 수 있음

## Draft Long List

### 1. Chloris gayana

- common route label:
  `Rhodes grass`
- why carry:
  - KEPHIS national variety list에 `Mbarara Rhodes`, `Boma Rhodes`, `Elmba Rhodes`가 있음
  - owner / maintainer / source 축에 `Kenya Seed Co.` 또는 `KARI/Kenya Seed Co.`가 적혀 formal route 설명이 쉬움
  - commercial/delivery lens에서 seed route가 가장 선명한 Chloris candidate임
  - genome lens에서는 이미 draft scaffold reference가 있으나, chromosome-grade saturation까지는 아님
- early risk:
  - `HF-3 novelty`에서 완전 무자원 후보보다 약할 수 있음
  - exact use-case를 generic hay/forage가 아니라 N1 시나리오 pain point에 맞게 더 좁혀야 함
- exemplar release lines:
  - `Mbarara Rhodes`
  - `Boma Rhodes`
  - `Elmba Rhodes`

### 2. Chloris roxburghiana

- common route label:
  `Horsetail grass`
- why carry:
  - KEPHIS national variety list에 `Chloris roxburghiana var. CHROX-KBK`가 있음
  - `KALRO` owner / maintainer / source가 명확함
  - KEPHIS 문서상 drought tolerance, ASAL adaptation, cut-and-carry or baling suitability가 적혀 있음
  - 현재 확보 근거 기준 public nuclear assembly가 보이지 않아 `SG-7 / HF-3` 방어가 더 강함
- early risk:
  - pure private direct-sales route보다 `public/blended` 쪽 설명력이 더 강함
  - legal route를 breeder-derived / source-holder 문서로 한 번 더 잠가야 함
- exemplar release line:
  - `CHROX-KBK`

## Reserve, Not Main Carry

### Cenchrus ciliaris

- 이유:
  - Kenya official list와 KVDA route가 선명하고 genome novelty도 강함
  - 하지만 current evidence 기준 polyploid/apomictic complexity 때문에 `HF-7 genome technical feasibility` 리스크가 더 큼
- 처리:
  `reserve candidate`

## Do Not Carry In This Draft

### Napier / Cenchrus purpureus

- 이유:
  public genome saturation이 높고, `reference-genome necessity` 방어가 약함

### generic Urochloa/Brachiaria

- 이유:
  commercial route는 강하지만 species-complex 차원의 public genome 자원이 이미 많아 broad subgroup으로는 carry 금지

## 다음 적용 순서

1. `SG-6`, `SG-8` 잠금
2. 이 draft를 formal long list로 승격할지 결정
3. 승격 시 `HF-1 ~ HF-9`를 species 단위로 적용
