# 시나리오 선정 기준 — 압축 하네스

문서 상태: COMMON
근거: `1차 사이클 결과/03_검증_ARCHIVE_치명오류시만사용`의 기존 체크리스트와 하네스를 압축 반영
적용 방식: 과도한 점수 계산은 제거하고, 발표 리포트에 필요한 항목만 유지

## 1. 기본 원칙

시나리오 선정은 후보를 끝없이 탈락시키는 검증 루프가 아니다. 작물 선정 과정을 리포트로 설득하기 위해 필요한 근거를 모으는 단계다.

기존 하네스에서 유지할 원칙:

- Hard Filter와 Soft Score의 구분
- 공식 출처와 anchor 사이의 fatal conflict만 중단 사유로 처리
- minor evidence gap, exact buyer/payer 미확정, 세부 accession 부족은 risk 또는 부록으로 이동
- D2/D4의 과도한 genome deep-dive는 블록 2로 넘김
- AHP/TPP는 본문 계산이 아니라 부록 또는 설명 배경으로만 사용

## 2. Hard Filter

다음 중 하나라도 해당하면 즉시 수정하거나 lead 후보에서 제외한다.

| 항목 | 질문 | 처리 |
|---|---|---|
| PBL 적합성 | 수요 증가 작물의 표준유전체 프로젝트로 설명 가능한가 | 불가능하면 제외 |
| 사실 충돌 | 공식 출처와 핵심 주장 사이에 정면 충돌이 있는가 | 즉시 수정 |
| 작물-시나리오 일치 | 작물의 실제 이용·재배·공급 구조가 시나리오와 맞는가 | 맞지 않으면 제외 |
| 유전체 명분 | 기존 자원 위에서 보강할 형질·지역·계통 명분이 있는가 | 없으면 제외 |
| 발표 가능성 | 30분 발표에서 설명 가능한가 | 불가능하면 reserve 또는 부록 |

## 3. Soft Score 축

점수 계산은 하지 않고, 각 축을 `강함 / 보통 / 약함`으로만 판정한다.

| 축 | 기존 하네스 대응 | 질문 | 본문 반영 |
|---|---|---|---|
| D1. 수요·시장 경로 | 시장·소비·정책 수요 | 누가 왜 이 작물을 필요로 하는가 | 작물 선정 근거 |
| D2. 유전체 명분 | reference/pangenome 공백 또는 보강 가치 | 기존 자원 위에서 무엇을 보강할 것인가 | 표준유전체 현황 |
| D3. 재배·형질 문제 | 채택·생산성·현장 문제 | 어떤 재배 난점이 유전체 분석 목표가 되는가 | 블록 2 질문 |
| D4. 실행 가능성 | genome size, ploidy, sequencing 난이도 | 지금 단계에서 설계 가능한가 | 데이터 전략 |
| D5. 공급·사업 연결 | buyer, payer, seed route, adoption | 유전체 자원이 공급·수익성으로 연결되는가 | 블록 3 활용 |

## 4. 발표용 판정 방식

| 판정 | 의미 | 처리 |
|---|---|---|
| Strong | 본문 lead 근거로 사용 가능 | 본문 슬라이드 |
| Moderate | 설명 가능하지만 리스크를 함께 표시 | 본문 또는 Q&A |
| Weak | 본문 lead 근거로 약함 | reserve 또는 부록 |
| Unknown | 지금 닫을 필요가 없는 미확정 | risk/assumption |
| Fatal | 주장 자체가 무너지는 충돌 | 즉시 수정 또는 제외 |

## 5. 제거한 과세부 항목

다음 항목은 기존 하네스에서 참조만 하고, 새 프로토콜에서는 본문 작업을 막지 않는다.

- AHP weight 재계산
- sensitivity analysis
- 모든 후보의 정량 시장 규모 재검증
- 모든 accession·source-holder 확정
- NCBI/Ensembl/논문 전체 수치 비교표
- 국가별 법률·계약 구조 세부 검토

## 6. 현재 후보 적용 메모

### India / Vigna mungo

- D1: 강함. APEDA와 PIB 자료로 pulses, Urad, MSP/procurement 경로가 설명된다.
- D2: 보통-강함. 공개 유전체가 있으므로 최초 구축이 아니라 disease/stress trait resource로 잡아야 한다.
- D3: 강함. MYMD, powdery mildew, drought, salinity, short-duration stability가 블록 2 질문으로 연결된다.
- D4: 보통. 기존 reference가 있어 설계는 가능하지만 novelty 표현을 조심해야 한다.
- D5: 강함. public route와 protein-security 서사가 선명하다.

### Kenya / Chloris gayana

- D1: 강함. dairy forage route와 quality fodder 수요가 설명된다.
- D2: 강함. draft genome과 SNP resource가 있어 고도화·활용 전략을 세우기 좋다.
- D3: 강함. forage yield, persistence, stress resilience, seed quality가 분석 질문으로 연결된다.
- D4: 보통-강함. diploid/tetraploid cultivar와 forage crop 특성을 고려해야 한다.
- D5: 보통-강함. dairy input과 seed pathway로 연결되지만 계약 수준 buyer 확정은 필요 없다.
