# 엔지니어링 가이드 인덱스

이 폴더는 프롬프트 설계, 사고 프레임, 하네스 생성에 직접 연결되는 가이드 문서를 모아둔 곳이다.

## 파일별 역할

- `meta-prompt-checklist-for-llm.md`: 프롬프트를 만들거나 고칠 때 빠뜨리기 쉬운 항목을 점검하는 체크리스트
- `meta-prompt-multi-agent-design-brief.md`: 멀티 에이전트 조사나 병렬 분업 설계를 짤 때 쓰는 역할 분해 가이드
- `prompt-engineering-guide.md`: 전반적인 프롬프트 엔지니어링 실전 가이드
- `prompt-engineering-tiers.md`: 프롬프트 품질을 단계별로 보는 기준표

## 언제 무엇을 먼저 볼까

- 새 프롬프트를 쓸 때: `meta-prompt-checklist-for-llm.md`
- 여러 조사 트랙을 병렬로 돌릴 때: `meta-prompt-multi-agent-design-brief.md`
- 프롬프트 구조를 전체적으로 손볼 때: `prompt-engineering-guide.md`
- 결과물 품질 기준을 맞추고 싶을 때: `prompt-engineering-tiers.md`
